# APPARATVS ROMANVS

Implementation of the Roman Numeral Calculator. Depends on the the [Check unit testing framework](https://libcheck.github.io/check/).

## Installation

     $> autoreconf --install
     $> ./configure
     $> make
     $> make check

## Usage

For addition:

    $> apparatus XI IX
       SVMMA: XX

For subtraction:

    $> apparatus -s XI IX
           SVMMA: II
